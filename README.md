# Micro-Project--II-Semester-IV-IIoT-
Hybrid IOT Temperature Monitoing System for Hydraulic Oil


📖 Project Description

This project is a Hybrid IoT Temperature Monitoring System for Hydraulic Oil.
It monitors the oil temperature in real-time using sensors and sends data through IoT, helping to prevent overheating and improve machine safety.

🔧 Components Used

- Temperature Sensor
- Microcontroller (ESP#32)
- relay
- LEDs
- BUZZER
- Push Button
- Fan
- Connecting Wires

⚙️ Working

The temperature sensor measures the hydraulic oil temperature and sends the data to the microcontroller.
The microcontroller processes the data and transmits it to an online platform using IoT for real-time monitoring.

📊 Output

- Displays real-time temperature readings
- Data can be monitored remotely
- Helps detect overheating early
